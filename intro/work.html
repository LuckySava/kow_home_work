А ви знаєте, що використовують Web Developer у своїй роботі?

HTML

  • Початок і фундамент будь-якого сайту. Основа основ
  • Самі великі проекти, починаються з елементарного створення каркасу на HTML
  • Сучасні Веб-розробники використовують HTML5 - останню специфікацію (версію), яка має багато інновацій в порівнянні з попередньою версією
  • HTML - не є мовою програмування, це мова розмітки

JavaScript

  • По-перше JavaScript - найпопулярніша мова програмування за версією GitHub
  • JavaScript робить сайт інтерактивним, дає логіку всім його елементам і взаємодію з користувачем
  • Колись це була мова тільки для Web, зараз JavaScript використовують для Back-End, створення ігор, робототехніка та багато іншого
  • Зараз використовують специфікацію ES6

React

  • Одна з найпопулярніших бібліотек JS, яка настільки стала "продвинутою", що багато хто, називає React фреймворком
  • React - розроблюється і підтримується Facebook та Instagram і там же використовується
  • React дає змогу створювати надшвидкі інтерактивні дії з сайтом, без перезавантаження сторінки
  • На React можна створювати HTML елементи і власне DOM дерево

NPM

  • Пакетний менеджер, який входить до складу Node.js
  • За допомогою NPM ми можемо значно спростити розробку, і скоротити час на написання коду, шляхом встановлення вже готового рішення
  • В NPM можна знайти пакети для вирішення майже любих завдань. При бажанні ви навіть можете знайти там пакет "АТБ" :)

CSS

  • Якщо HTML - це каркас, то CSS це "краса" і вигляд сайту
  • CSS - "мова" або ж краще сказати таблиця стилів
  • Жоден сайт в Інтернеті не обходиься без CSS
  • Сучасні розробники використовують так звані "препроцесори", які значно спрощують написання коду, роблячи його більш зрозумілим і впорядкованим

PhpStorm

  • Це комерційне середовище розробки для програмістів на різних мовах програмування
  • Має зручний інтерфейс, що пришвидшує написання коду та знаходження помилок в ньому
  • Розроблена міжнародною компанією JetBrains
